# Cròniques de Catalunya RPG

RPG offline per aprendre català jugant. PWA instal·lable sense internet.

## Què inclou
- 3 capítols: Barcelona, Girona, València
- Sistema de llegendes desbloquejables
- Minijoc de frases amb emojis
- Botiga i biblioteca d'emojis
- Personatges i sistema de tòtems

## Instal·lar
1. Obre el link de GitHub Pages al mòbil
2. Chrome/Safari → "Afegeix a la pantalla d'inici"
3. Juga 100% offline

## Tech
HTML, CSS, JS pur + PWA + Service Worker. Sense frameworks.
